# 2.计算机体系结构
## 冯诺依曼体系结构：  ![冯诺依曼体系结构示意图](./images/冯诺依曼体系结构示意图.jpg)  
### 控制器主要组成部件：  
1. 程序计数器(pc) -> 存放下一条指令的地址  
2. 指令寄存器(IR) -> 存放当前正在执行的指令
3. 指令译码器(ID) -> 分析指令中的操作码
4. 地址寄存器(AR)
5. 时序部件
### 运算器主要组成部分：
1. 算术逻辑单元(ALU) -> 执行算术运算何逻辑运算
2. 累加寄存器(AC) -> 暂时存放运算的中间结果
3. 数据缓冲寄存器
4. 状态条件寄存器
5. 状态条件寄存器
### 存储器
分主存储器(主存或内存)和辅助存储器(辅存或外存)，和CPU直接交换信息的是主存。

## 计算机总线
1. 定义：是计算机各种功能部件之间传送信息的公共通信干线。  
2. 按所传输的信息种类划分：数据总线、地址总线、控制总线 。
-  数据总线宽度：指CPU中运算器与存储器之间互连的内部总线根数，也指CPU一次读取的二进制位数，同时也说明字长。
- 地址总线的宽度：CPU寻址范围，若地址总线为n位，则可寻址空间为2的n次方。
3. 按传输方法划分：串行总线、并行总线
- 串行：USB、SATA、I2C
- 并行：PCI总线、IEEE488  
4. 32位操作系统理论上支持的最大内存空间容量为(4GB)<br>解析：此处的32位为地址总线宽度,寻址单元以字节为单位。
